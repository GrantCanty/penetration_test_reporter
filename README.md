# Pen Writer  
Pen Writer is a CLI meant to automate penetration testing and create a 1 page report using an LLM.  

## Requirements:  
* Clone the following Github repository: https://github.com/OWASP/Nettacker.git  
* Build the required Docker image for the cloned repository with: docker build -t nettacker .
